About the Project

Appetite is a fully functional, cross-platform application that seamlessly combines recipe management with social networking functionality. Winner of the 2023 Utah TSA State Software Development Competition, Appetite provides users with a modern, intuitive interface to store, discover, and share their favorite recipes while connecting with a community of food enthusiasts.

Built with performance and user experience as top priorities, Appetite works flawlessly on iOS, Android, and web platforms, delivering a consistent, fast experience across all devices.

Recognition & Awards

1st Place Winner

2023 Utah TSA State Software Development Competition

Appetite was recognized for excellence in software design, functionality, and user experience. The project demonstrated strong technical implementation, innovative features, and polished user interface design that set it apart from competing submissions.

Core Capabilities

Recipe Management System

Comprehensive recipe storage and organization with advanced search capabilities. Users can create, edit, and categorize recipes with detailed ingredient lists, preparation steps, and nutritional information.

Social Network Integration

Built-in social features enable users to build profiles, follow other users, and share recipes within a community. Real-time feed displays recent activity and trending recipes from connected users.

Discovery & Search

Intelligent recipe discovery system with filtering, tagging, and recommendation algorithms. Users can explore new recipes based on ingredients, cuisine type, dietary restrictions, and preparation time.

Cross-Platform Architecture

Native mobile applications for iOS and Android with web support. Seamless data synchronization ensures recipes and user data remain consistent across all platforms and devices.

Performance Optimization

Engineered for speed with efficient data caching, lazy loading, and optimized rendering. The application maintains smooth performance even with large recipe collections and extensive social feeds.

Modern User Interface

Clean, intuitive design following platform-specific UI guidelines. Responsive layouts adapt seamlessly to different screen sizes while maintaining consistent branding and user experience.

Technical Implementation

Built with modern technologies prioritizing developer experience, code maintainability, and scalable architecture.
